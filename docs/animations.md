# Plan d'animations

## Global
- Scroll-reveal: fade + slide doux (600ms) sur sections.
- Hover cards: léger lift + glow subtil.
- Navbar: blur léger en sticky.
- Respect prefers-reduced-motion.

## Hero
- Gradient doux + grain léger (background image noise).
- Search bar: focus visible + micro glow.

## Offres
- Cards: parallax très discret (hover).
- CTA boutons: transition couleur 200ms.

## Match J.A.S
- Score animé (barre progressive).
- Boutons préférences: state fill.

## Listing
- Skeleton loading.
- Empty state: fade-in.

## Mobile
- Menu bottom sheet: slide-up rapide (250ms).
