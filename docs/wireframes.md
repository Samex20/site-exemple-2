# Wireframe narratif par page

## Accueil
- Header sticky + CTA “Trouver un job”.
- Hero: H1 fort + search bar premium + filtres rapides.
- Bloc métriques + photo éditoriale.
- Comment ça marche (3 cards).
- Offres vedettes (3 cards).
- Coaching (packs + témoignages).
- Employeurs (value props + mini pricing).
- CTA final + taglines.

## Emplois
- Titre + description.
- Filtres rapides + chips.
- Toggle grille/liste.
- Résultats (cards) avec pagination.
- Upsell coaching en bas.
- États: loading skeleton, empty state.

## Fiche emploi
- Hero info (titre, salaire, tags).
- Description + tâches.
- Checklist compétences.
- CTA Postuler + Favoris.
- Match J.A.S (sidebar).
- CTA coaching contextuel.
- Offres similaires.

## Coaching
- Hero + bénéfices.
- Grille packs (pricing cards).
- Exemples avant/après.
- FAQ.
- CTA “Parler à un coach”.

## Ressources
- Grille articles/templates par catégorie.
- Lead magnet (email + bouton).

## Espace étudiant
- Widgets stats.
- Table suivi candidatures.
- Liste des actions prioritaires.

## Employeurs
- Hero + preuves.
- Tarifs (3 cards).
- Formulaire publier une offre.
- Qualité + FAQ.

## À propos / Contact / Légal
- Pages simples, informatives, orientées conversion.
