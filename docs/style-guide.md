# Style Guide J.A.S

## Direction artistique
- Style: blanc premium, espaces généreux, surfaces vitrées subtiles, micro-détails fins.
- Photographie: étudiants au Québec, rendu éditorial, lumière naturelle, pas cheesy.
- Icônes: line icons minimalistes, stroke 1.5-2px, angles doux.

## Typographie
- Display: Sora (titres, CTA clés)
- Body: Manrope (texte courant, microcopy)
- Hiérarchie: H1 48-56px, H2 32-36px, H3 20-24px, body 16-18px

## Palettes
### Palette A (premium doux)
- Fond: #f9f9f7 (frost)
- Blanc cassé: #f6f5f2 (mist)
- Noir doux: #0e1116 (ink)
- Gris neutre: #3a3f46
- Accent: #1f7a8c (bleu/vert)

### Palette B (punchy élégant)
- Fond: #ffffff
- Graphite: #1f2328
- Gris: #3a3f46
- Accent corail: #ff5d5d
- Accent électrique (option): #2f6bff

## Spacing & layout
- Grille: 12 colonnes, max-width 1120-1200px
- Radius: 16-24px
- Section spacing: 64-96px
- Padding cards: 24-32px

## Composants
- Navbar sticky + blur
- Search bar premium (inputs + bouton plein)
- Cards (job, coaching, témoignage)
- Tags pills (uppercase tracking)
- Pricing cards (employeurs + coaching)
- Accordions FAQ
- États (loading, empty, error, success)

## Accessibilité
- Contraste AA minimum
- Focus visible (ring + outline)
- Labels pour inputs
- Respect prefers-reduced-motion
